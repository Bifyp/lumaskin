# Інструкція для користувача - Luma Skin Laser Studio

## Вхід в адмін-панель

1. Перейдіть на [https://yourdomain.com/admin](https://yourdomain.com/admin)
2. Введіть email та пароль
3. Натисніть "Вхід"

## Управління услугами

### Додати услугу
1. **Admin** → **Services**
2. Натисніть **"+ Add Service"**
3. Заповніть форму:
   - Назва услуги
   - Опис
   - Ціна
   - Тривалість (хвилини)
4. Натисніть **"Save"**

### Редагувати услугу
1. Перейдіть до услуги
2. Натисніть **"Edit"**
3. Змініть дані
4. Натисніть **"Save"**

### Видалити услугу
1. Натисніть на услугу
2. Натисніть **"Delete"**
3. Підтвердіть видалення

## Управління пакетами

### Додати пакет
1. **Admin** → **Packages**
2. Натисніть **"+ Add Package"**
3. Заповніть:
   - Назва пакета
   - Описання
   - Ціна
   - Кількість сеансів
4. Натисніть **"Save"**

## Управління галереєю

### Додати фото
1. **Admin** → **Gallery**
2. Натисніть **"+ Upload Photo"**
3. Виберіть зображення з комп'ютера
4. Додайте описання (опціонально)
5. Натисніть **"Upload"**

### Видалити фото
1. Наведіть мишку на фото
2. Натисніть **"Delete"**
3. Підтвердіть видалення

## Перегляд букінгів

1. **Admin** → **Bookings**
2. Виберіть букінг зі списку
3. Переглядайте:
   - Ім'я клієнта
   - Email та телефон
   - Обрану услугу
   - Дату та час
   - Статус

### Змінити статус букінга
1. Відкрийте букінг
2. Натисніть на статус (Pending → Confirmed → Completed)
3. Зберегти

## Управління перекладами контенту

1. **Admin** → **Translations**
2. Виберіть сторінку для редагування:
   - Home
   - About
   - Services
   - Contact
   - тощо
3. Змініть текст
4. Натисніть **"Save"**

> 📝 **Примітка:** Текст автоматично перекладається на інші мови за допомогою DeepL

## Часті питання

**Як додати новий номер телефону для зв'язку?**
- Редагуйте контакти в **Admin** → **Translations** → **Contact Page**

**Як зміниться сайт після того як я щось відредагую?**
- Зміни з'являються миттєво на публічному сайті

**Мій букінг не надійшов - що робити?**
- Перевірте папку "Spam" у своїй пошті
- Перевірте налаштування email в админ-панелі

**Забув пароль**
- Натисніть **"Forgot password"** на сторінці входу
- Введіть email та дотримуйтесь інструкцій
